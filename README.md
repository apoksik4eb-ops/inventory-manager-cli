Inventory Manager CLI

Консольное приложение для управления складом, разработанное на Python с использованием SQLAlchemy и PostgreSQL.

Возможности

Категории:
Создание категории
Просмотр списка категорий
Получение категории по ID
Изменение названия категории
Удаление категории

Поставщики:
Создание поставщика
Просмотр списка поставщиков
Получение поставщика по ID
Изменение контактных данных
Деактивация поставщика
Удаление поставщика

Товары:
Создание товара
Просмотр списка товаров
Получение товара по ID
Обновление цен
Изменение минимального остатка
Деактивация товара
Удаление товара

Складские операции:
Поступление товара
Списание товара
Корректировка остатка
Просмотр истории операций

Отчёты:
Товары с категориями и поставщиками
История операций с названиями товаров
Товары по категории
Товары по поставщику
Операции по конкретному товару
Количество товаров по категориям
Количество товаров по поставщикам
Текущие остатки товаров
Товары с низким остатком
Общая закупочная стоимость склада
Потенциальная выручка склада
Потенциальная прибыль склада

Установка:
Создать виртуальное окружение:
python -m venv venv

Активировать виртуальное окружение:
venv\Scripts\activate

Установить зависимости:
pip install -r requirements.txt

Настроить подключение к PostgreSQL в файле database.py.
Создание таблиц

Таблицы создаются автоматически при запуске приложения:

python main.py
Заполнение тестовыми данными

Для загрузки тестовых данных выполнить:

python seed.py
